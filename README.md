# 1. What is Git?

Git is a **version control system** used by developers to track changes in their code.

It allows multiple developers to work on the same project without overwriting each other's work.

With Git you can:
- Track code changes
- Revert to older versions
- Collaborate with other developers
- Manage project history

Git works locally on your computer.

---

# 2. What is GitHub?

GitHub is a **cloud platform where Git repositories are stored**.

It allows developers to:
- Share their code
- Collaborate with others
- Manage projects
- Track issues

GitHub uses Git as its underlying system.
